# ユーザーメッセージの LaTeX レンダリング

::: info
**注意**: この機能はバージョン 1.3.7 以降でサポートされています。
:::

ユーザーメッセージ内の LaTeX 数式を自動的に数学記号としてレンダリングします。

## 概要

Gemini™ のチャット入力で LaTeX 構文（例：`$E=mc^2$` や `$$\int_0^1 x^2 dx$$`）を使用すると、Voyager が自動的に検出し、美しい数式として表示します。

### 主な機能

- **インライン数式**：`$...$` で囲む（例：`$x^2 + y^2 = r^2$`）
- **ディスプレイ数式**：`$$...$$` で囲むと、中央揃えのブロック表示
- **自動検出**：既存メッセージと新規送信メッセージを自動処理
- **KaTeX エンジン**：KaTeX による高品質な数式レンダリング

## 使い方

1. Gemini のチャット入力欄に LaTeX 構文を含むメッセージを入力
2. 送信後、数式が自動的に数学記号としてレンダリングされます
3. インライン数式 `$...$` はテキストの流れの中に表示
4. ディスプレイ数式 `$$...$$` は中央揃えで独立した行に表示

## 構文例

| 入力                                      | 説明             |
| ----------------------------------------- | ---------------- |
| `$E=mc^2$`                                | インライン数式   |
| `$$\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$$` | ディスプレイ数式 |
| `$\alpha, \beta, \gamma$`                 | ギリシャ文字     |
| `$\sqrt{x^2 + y^2}$`                      | 平方根           |

## 注意事項

::: tip ヒント

- ユーザーが送信したメッセージにのみ適用されます。Gemini の返信の数式は Gemini 自体がレンダリングします。
- LaTeX 構文にエラーがある場合、元のテキストがそのまま表示されます。
  :::
