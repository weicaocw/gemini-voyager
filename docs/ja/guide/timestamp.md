# メッセージタイムスタンプ

::: info
**注意**: この機能はバージョン 1.3.7 以降でサポートされています。
:::

Voyager は各メッセージの送信時刻を自動的に記録し、会話の全体的な流れを振り返るのに役立ちます。

## 機能紹介

Gemini はメッセージの送信時刻をネイティブで表示しません。会話が長くなると、特定の議論がいつ行われたか思い出すのが難しくなります。

Voyager は各メッセージに正確なタイムスタンプ（形式：`YYYY-MM-DD HH:mm:ss`）を自動的に表示し、タイムラインを一目で把握できるようにします。

## 機能の特徴

- **自動記録**：メッセージ送信時に自動的に時刻をキャプチャ、手動操作は不要です。
- **秒単位の精度**：完全な日付と時刻で、長い会話のレビューに最適です。
- **ローカル永続化**：タイムスタンプはローカルストレージに保存され、ページ更新後も表示されます。
- **ゼロ干渉**：時間情報は控えめに表示され、読書体験に影響を与えません。

## 利用シーン

- **作業記録**：特定の決定がいつ行われたかを振り返る。
- **学習ノート**：概念がいつ議論されたかを記録し、後の復習に役立てる。
- **長い会話のレビュー**：複数日にわたる深い会話で、異なる時間帯の議論を明確に区別する。
