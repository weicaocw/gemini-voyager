# フォントサイズ

::: warning
**近日公開**：この機能は次のバージョンで利用可能になります。お楽しみに。
:::

Gemini のチャットエリアのフォントサイズを自由に調整し、最も快適な読書体験を実現します。

## 機能紹介

フォントサイズの好みは人それぞれです——情報密度を高めるためにコンパクトなテキストを好む人もいれば、目の疲れを軽減するために大きなテキストを好む人もいます。Voyager では、チャットのフォントサイズを 80% から 150% の間で自由に調節できます。

## 機能の特徴

- **柔軟な範囲**：80%–150% のフォントサイズ調整をサポートし、ほとんどの読書ニーズをカバーします。
- **全面的なカバー**：ユーザーメッセージとモデル応答の両方に適用され、本文、リスト、テーブル、コードブロックを含みます。
- **コードブロック最適化**：コードブロックのフォントサイズは自動的に比例縮小（本文フォント × 0.875）し、コードと本文の視覚的階層を維持します。
- **即時反映**：調整後すぐに反映され、ページの更新は不要です。

## 使い方

1. Voyager の設定パネル（ポップアップ）を開きます。
2. **フォントサイズ** 設定を見つけ、スイッチをオンにします。
3. スライダーをドラッグしてフォントサイズの割合を調整します——ページがリアルタイムで変化をプレビューします。
