# Marca de Tiempo

::: info
**Nota**: Esta función está disponible en la versión 1.3.7 y posteriores.
:::

Voyager registra automáticamente la hora de envío de cada mensaje, ayudándote a rastrear la línea temporal completa de tus conversaciones.

## Introducción

Gemini no muestra de forma nativa la hora de envío de los mensajes. A medida que las conversaciones se alargan, se hace difícil recordar cuándo tuvo lugar una discusión particular.

Voyager anota cada mensaje con una marca de tiempo precisa (formato: `AAAA-MM-DD HH:mm:ss`), haciendo que la línea temporal sea clara de un vistazo.

## Características

- **Registro automático**: Las marcas de tiempo se capturan automáticamente al enviar — sin necesidad de acción manual.
- **Precisión al segundo**: Fecha y hora completas, perfecto para revisar conversaciones largas.
- **Persistencia local**: Las marcas de tiempo se guardan en el almacenamiento local y permanecen visibles tras actualizar la página.
- **Cero distracción**: La información temporal se presenta de forma discreta sin afectar la experiencia de lectura.

## Casos de uso

- **Registros de trabajo**: Revisar cuándo se alcanzó una decisión particular.
- **Notas de estudio**: Marcar cuándo se discutió un concepto para repasarlo después.
- **Revisión de conversaciones largas**: En conversaciones profundas de varios días, distinguir claramente las discusiones de diferentes períodos.
