# عرض LaTeX في رسائل المستخدم

::: info
**ملاحظة**: هذه الميزة مدعومة في الإصدار 1.3.7 وما بعده.
:::

عرض صيغ LaTeX في رسائل المستخدم تلقائياً كرموز رياضية.

## نظرة عامة

عند استخدام صيغة LaTeX في دردشة Gemini™ (مثل `$E=mc^2$` أو `$$\int_0^1 x^2 dx$$`)، يكتشف Voyager الصيغ ويعرضها تلقائياً كمعادلات رياضية جميلة.

### الميزات الرئيسية

- **صيغة سطرية**: غلّف بـ `$...$` (مثل `$x^2 + y^2 = r^2$`)
- **صيغة مستقلة**: غلّف بـ `$$...$$` لعرض متمركز في سطر منفصل
- **اكتشاف تلقائي**: يعالج الرسائل الموجودة والمرسلة حديثاً
- **محرك KaTeX**: عرض رياضي عالي الجودة بواسطة KaTeX

## طريقة الاستخدام

1. اكتب رسالة تحتوي على صيغة LaTeX في دردشة Gemini
2. بعد الإرسال، يتم عرض الصيغ تلقائياً كرموز رياضية
3. الصيغ السطرية `$...$` تظهر ضمن النص
4. الصيغ المستقلة `$$...$$` تظهر متمركزة في سطر منفصل

## أمثلة على الصيغ

| الإدخال                                   | الوصف        |
| ----------------------------------------- | ------------ |
| `$E=mc^2$`                                | صيغة سطرية   |
| `$$\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$$` | صيغة مستقلة  |
| `$\alpha, \beta, \gamma$`                 | أحرف يونانية |
| `$\sqrt{x^2 + y^2}$`                      | جذر تربيعي   |

## ملاحظات

::: tip تلميح

- ينطبق فقط على الرسائل المرسلة من المستخدم. صيغ ردود Gemini يعرضها Gemini نفسه.
- إذا كانت صيغة LaTeX غير صالحة، سيظهر النص الأصلي.
  :::
