# Message Timestamps

::: info
**Note**: This feature is available in version 1.3.7 and later.
:::

Voyager automatically records the send time of each message, helping you trace the full timeline of your conversations.

## Introduction

Gemini doesn't natively display when messages were sent. As conversations grow longer, it becomes difficult to recall when a particular discussion took place.

Voyager annotates each message with a precise timestamp (format: `YYYY-MM-DD HH:mm:ss`), making the timeline crystal clear.

## Features

- **Automatic Recording**: Timestamps are captured automatically when messages are sent — no manual action needed.
- **Second-level Precision**: Full date and time, perfect for reviewing long conversations.
- **Local Persistence**: Timestamps are saved in local storage and remain visible after page refresh.
- **Zero Distraction**: Time information is presented unobtrusively without affecting the reading experience.

## Use Cases

- **Work Records**: Review when a particular decision was reached.
- **Study Notes**: Mark when a concept was discussed for later review.
- **Long Conversation Review**: In multi-day deep conversations, clearly distinguish discussions from different time periods.
