# Поведение отправки

::: info
**Примечание**: Эта функция доступна в версии 1.3.7 и выше.
:::

Измените сочетание клавиш для отправки в Gemini на `Ctrl+Enter` (`⌘+Enter` на macOS), превратив `Enter` в клавишу переноса строки для спокойного написания многострочных промптов.

## Зачем эта функция

По умолчанию нажатие `Enter` в Gemini немедленно отправляет сообщение. При написании длинных промптов, вставке многострочного кода или аккуратном форматировании ввода это часто приводит к случайной отправке — вы хотели только новую строку, но сообщение уже улетело.

При включении:

- **Enter** → Вставка новой строки
- **Ctrl+Enter** (macOS: **⌘+Enter**) → Отправка сообщения

## Как включить

1. Откройте панель настроек Voyager (всплывающее окно).
2. Найдите переключатель **Ctrl+Enter для отправки**.
3. Включите его.

## Особенности

- **Нулевая нагрузка на производительность**: При отключении слушатели событий клавиатуры не активны — никакого влияния на производительность страницы.
- **Мгновенное переключение**: Изменения вступают в силу сразу после переключения в настройках, без обновления страницы.
- **Совместимость с режимом редактирования**: Работает и при редактировании ранее отправленных сообщений.
