# Рендеринг LaTeX в сообщениях пользователя

::: info
**Примечание**: Эта функция поддерживается в версии 1.3.7 и более поздних.
:::

Автоматический рендеринг формул LaTeX в сообщениях пользователя в виде математических символов.

## Обзор

Когда вы используете синтаксис LaTeX в чате Gemini™ (например, `$E=mc^2$` или `$$\int_0^1 x^2 dx$$`), Voyager автоматически обнаруживает и отображает красивые математические формулы.

### Основные возможности

- **Строчная формула**: оберните в `$...$` (например, `$x^2 + y^2 = r^2$`)
- **Блочная формула**: оберните в `$$...$$` для отображения по центру на отдельной строке
- **Автообнаружение**: обрабатывает существующие и вновь отправленные сообщения
- **Движок KaTeX**: высококачественный рендеринг математики с помощью KaTeX

## Как использовать

1. Введите сообщение с синтаксисом LaTeX в чате Gemini
2. После отправки формулы автоматически отображаются как математические символы
3. Строчные формулы `$...$` отображаются в потоке текста
4. Блочные формулы `$$...$$` отображаются по центру на отдельной строке

## Примеры синтаксиса

| Ввод                                      | Описание          |
| ----------------------------------------- | ----------------- |
| `$E=mc^2$`                                | Строчная формула  |
| `$$\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$$` | Блочная формула   |
| `$\alpha, \beta, \gamma$`                 | Греческие буквы   |
| `$\sqrt{x^2 + y^2}$`                      | Квадратный корень |

## Примечания

::: tip Подсказка

- Применяется только к сообщениям, отправленным пользователем. Формулы в ответах Gemini рендерятся самим Gemini.
- Если синтаксис LaTeX недействителен, будет показан исходный текст.
  :::
