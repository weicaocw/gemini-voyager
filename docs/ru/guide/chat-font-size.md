# Размер шрифта

::: warning
**Скоро**: Эта функция будет доступна в следующей версии. Следите за обновлениями.
:::

Свободно настраивайте размер шрифта в области чата Gemini для максимально комфортного чтения.

## Введение

У каждого свои предпочтения по размеру шрифта — кто-то предпочитает компактный текст для большей плотности информации, а кто-то — крупный текст для снижения усталости глаз. Voyager позволяет регулировать размер шрифта чата от 80% до 150%.

## Особенности

- **Гибкий диапазон**: Поддержка регулировки размера шрифта от 80% до 150%, покрывающая большинство потребностей в чтении.
- **Полное покрытие**: Применяется как к сообщениям пользователя, так и к ответам модели, включая основной текст, списки, таблицы и блоки кода.
- **Оптимизация блоков кода**: Размер шрифта блоков кода автоматически пропорционально уменьшается (шрифт текста × 0,875), сохраняя визуальную иерархию между кодом и прозой.
- **Мгновенный эффект**: Изменения вступают в силу немедленно без обновления страницы.

## Как использовать

1. Откройте панель настроек Voyager (всплывающее окно).
2. Найдите настройку **Размер шрифта** и включите её.
3. Перетащите ползунок для регулировки процента размера шрифта — страница покажет изменения в реальном времени.
