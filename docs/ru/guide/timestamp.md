# Временные метки сообщений

::: info
**Примечание**: Эта функция доступна в версии 1.3.7 и выше.
:::

Voyager автоматически записывает время отправки каждого сообщения, помогая вам отслеживать полную хронологию ваших разговоров.

## Введение

Gemini не отображает время отправки сообщений. По мере удлинения разговоров становится трудно вспомнить, когда произошла та или иная дискуссия.

Voyager добавляет к каждому сообщению точную временную метку (формат: `ГГГГ-ММ-ДД ЧЧ:мм:сс`), делая хронологию наглядной с первого взгляда.

## Особенности

- **Автоматическая запись**: Временные метки фиксируются автоматически при отправке — никаких ручных действий не требуется.
- **Точность до секунды**: Полная дата и время, идеально для просмотра длинных разговоров.
- **Локальное хранение**: Метки сохраняются в локальном хранилище и остаются видимыми после обновления страницы.
- **Без отвлечений**: Временная информация отображается ненавязчиво, не влияя на чтение.

## Сценарии использования

- **Рабочие записи**: Просмотр того, когда было принято определённое решение.
- **Учебные заметки**: Отметка времени обсуждения концепции для последующего повторения.
- **Обзор длинных разговоров**: В глубоких разговорах на протяжении нескольких дней чётко различать обсуждения из разных временных периодов.
