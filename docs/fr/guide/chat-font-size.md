# Taille de Police

::: warning
**Bientôt disponible** : Cette fonctionnalité sera disponible dans la prochaine version. Restez à l'écoute.
:::

Ajustez librement la taille de police de la zone de chat de Gemini pour l'expérience de lecture la plus confortable.

## Introduction

Chacun a des préférences différentes en matière de taille de police — certains préfèrent un texte compact pour une densité d'information accrue, tandis que d'autres préfèrent un texte plus grand pour réduire la fatigue visuelle. Voyager vous permet d'ajuster la taille de police du chat entre 80% et 150%.

## Caractéristiques

- **Plage flexible** : Prend en charge l'ajustement de la taille de police de 80% à 150%, couvrant la plupart des besoins de lecture.
- **Couverture complète** : S'applique aux messages utilisateur et aux réponses du modèle, y compris le texte, les listes, les tableaux et les blocs de code.
- **Optimisation des blocs de code** : La taille de police des blocs de code est automatiquement réduite proportionnellement (police du texte × 0,875), maintenant la hiérarchie visuelle entre code et prose.
- **Effet instantané** : Les changements prennent effet immédiatement sans actualisation de page.

## Comment utiliser

1. Ouvrez le panneau de paramètres de Voyager (popup).
2. Trouvez le paramètre **Taille de Police** et activez-le.
3. Faites glisser le curseur pour ajuster le pourcentage de taille de police — la page prévisualisera les changements en temps réel.
