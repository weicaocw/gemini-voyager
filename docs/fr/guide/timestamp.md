# Horodatage des Messages

::: info
**Note** : Cette fonctionnalité est disponible à partir de la version 1.3.7.
:::

Voyager enregistre automatiquement l'heure d'envoi de chaque message, vous aidant à retracer le fil complet de vos conversations.

## Introduction

Gemini n'affiche pas nativement l'heure d'envoi des messages. Lorsque les conversations s'allongent, il devient difficile de se rappeler quand une discussion particulière a eu lieu.

Voyager annote chaque message avec un horodatage précis (format : `AAAA-MM-JJ HH:mm:ss`), rendant la chronologie parfaitement lisible.

## Caractéristiques

- **Enregistrement automatique** : Les horodatages sont capturés automatiquement lors de l'envoi — aucune action manuelle requise.
- **Précision à la seconde** : Date et heure complètes, idéal pour revoir de longues conversations.
- **Persistance locale** : Les horodatages sont sauvegardés dans le stockage local et restent visibles après actualisation de la page.
- **Zéro distraction** : L'information temporelle est présentée discrètement sans affecter l'expérience de lecture.

## Cas d'utilisation

- **Historique de travail** : Revoir quand une décision particulière a été prise.
- **Notes d'étude** : Marquer quand un concept a été discuté pour une révision ultérieure.
- **Revue de longues conversations** : Dans des conversations approfondies sur plusieurs jours, distinguer clairement les discussions de différentes périodes.
