# 字體大小

::: warning
**即將推出**：該功能將在下個版本中上線，敬請期待。
:::

自由調整 Gemini 對話區域的字體大小，打造最舒適的閱讀體驗。

## 功能介紹

每個人對字體大小的偏好不同——有人喜歡緊湊的小字以獲取更多資訊密度，有人偏好大字以減輕閱讀疲勞。Voyager 讓你可以在 80% 到 150% 之間自由調節對話區域的字號。

## 功能特點

- **範圍靈活**：支援 80%–150% 的字號調節，涵蓋絕大多數閱讀需求。
- **全面覆蓋**：同時作用於使用者訊息和模型回覆，包括正文、列表、表格和程式碼區塊。
- **程式碼區塊最佳化**：程式碼區塊字號會自動按比例縮小（正文字號 × 0.875），保持程式碼與正文的視覺層次。
- **即時生效**：調整後立即生效，無需重新整理頁面。

## 如何使用

1. 打開 Voyager 的設定面板（彈窗）。
2. 找到 **字體大小** 設定項，開啟開關。
3. 拖動滑桿調節字號百分比，頁面會即時預覽效果。
