# 傳送行為

::: info
**注意**：該功能在 1.3.7 及後續版本中支援。
:::

將 Gemini 的傳送快捷鍵改為 `Ctrl+Enter`（macOS 下為 `⌘+Enter`），`Enter` 鍵變為換行，讓你在撰寫多行提示詞時更加從容。

## 為什麼需要這個功能

Gemini 預設按下 `Enter` 即傳送訊息。在編寫較長的 prompt、貼上多行程式碼、或需要精心排版時，經常導致誤發——你只想換個行，訊息卻已經飛出去了。

開啟後：

- **Enter** → 插入換行
- **Ctrl+Enter**（macOS: **⌘+Enter**） → 傳送訊息

## 如何開啟

1. 打開 Voyager 的設定面板（彈窗）。
2. 找到 **Ctrl+Enter 傳送** 開關。
3. 打開開關即可生效。

## 功能特點

- **零效能開銷**：關閉時不監聽任何鍵盤事件，不影響頁面效能。
- **即時切換**：在設定中開關後立即生效，無需重新整理頁面。
- **相容編輯模式**：在編輯已傳送的訊息時同樣適用。
