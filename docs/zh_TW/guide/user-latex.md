# 使用者訊息 LaTeX 渲染

::: info
**注意**：該功能僅在 1.3.7 及後續版本中支援。
:::

自動將使用者訊息中的 LaTeX 公式渲染為數學符號。

## 功能介紹

當你在 Gemini™ 的聊天輸入框中使用 LaTeX 語法（如 `$E=mc^2$` 或 `$$\int_0^1 x^2 dx$$`），Voyager 會自動偵測並渲染為美觀的數學公式。

### 主要特性

- **行內公式**：使用 `$...$` 包裹，如 `$x^2 + y^2 = r^2$`
- **獨立公式**：使用 `$$...$$` 包裹，公式獨佔一行顯示
- **自動偵測**：自動處理頁面上已有的和新發送的訊息
- **KaTeX 渲染**：使用 KaTeX 引擎進行高品質數學公式渲染

## 使用方法

1. 在 Gemini 聊天輸入框中輸入包含 LaTeX 語法的訊息
2. 發送後，公式會自動渲染為數學符號
3. 行內公式 `$...$` 與文字同行顯示
4. 獨立公式 `$$...$$` 會置中獨佔一行

## 語法範例

| 輸入                                      | 說明     |
| ----------------------------------------- | -------- |
| `$E=mc^2$`                                | 行內公式 |
| `$$\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$$` | 獨立公式 |
| `$\alpha, \beta, \gamma$`                 | 希臘字母 |
| `$\sqrt{x^2 + y^2}$`                      | 根號     |

## 注意事項

::: tip 提示

- 僅對使用者發送的訊息生效，Gemini 回覆中的公式由 Gemini 自身渲染。
- 如果 LaTeX 語法有誤，將顯示原始文字。
  :::
