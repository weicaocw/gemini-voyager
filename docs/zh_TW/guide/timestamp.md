# 訊息時間戳

::: info
**注意**：該功能在 1.3.7 及後續版本中支援。
:::

Voyager 會自動為每則訊息記錄傳送時間，幫助你回溯對話的完整脈絡。

## 功能介紹

Gemini 原生不顯示訊息的傳送時間，當對話變長後，很難回憶某段討論發生在何時。

Voyager 在每則訊息旁自動標註精確的傳送時間戳（格式：`YYYY-MM-DD HH:mm:ss`），讓時間線一目了然。

## 功能特點

- **自動記錄**：傳送訊息時自動捕獲時間，無需手動操作。
- **精確到秒**：完整的日期與時間，適合複盤長對話。
- **本地持久化**：時間戳保存在本地儲存中，重新整理頁面後依然可見。
- **零干擾**：時間資訊以簡潔的方式呈現，不影響對話閱讀體驗。

## 使用場景

- **工作記錄**：回顧某個決策是在什麼時間點達成的。
- **學習筆記**：標記某個知識點是何時討論的，方便後續複習。
- **長對話複盤**：在跨天的深度對話中，清晰區分不同時間段的討論內容。
