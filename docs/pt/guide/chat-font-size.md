# Tamanho da Fonte

::: warning
**Em breve**: Esta funcionalidade estará disponível na próxima versão. Fique atento.
:::

Ajuste livremente o tamanho da fonte da área de chat do Gemini para a experiência de leitura mais confortável.

## Introdução

Cada pessoa tem preferências diferentes quanto ao tamanho da fonte — alguns preferem texto compacto para maior densidade de informação, enquanto outros preferem texto maior para reduzir a fadiga visual. O Voyager permite ajustar o tamanho da fonte do chat entre 80% e 150%.

## Características

- **Gama flexível**: Suporta ajuste de tamanho de fonte de 80% a 150%, cobrindo a maioria das necessidades de leitura.
- **Cobertura completa**: Aplica-se tanto às mensagens do utilizador como às respostas do modelo, incluindo texto, listas, tabelas e blocos de código.
- **Otimização de blocos de código**: O tamanho da fonte dos blocos de código é automaticamente reduzido proporcionalmente (fonte do texto × 0,875), mantendo a hierarquia visual entre código e prosa.
- **Efeito instantâneo**: As alterações entram em vigor imediatamente sem atualização da página.

## Como usar

1. Abra o painel de definições do Voyager (popup).
2. Encontre a definição de **Tamanho da Fonte** e ative-a.
3. Arraste o cursor para ajustar a percentagem do tamanho da fonte — a página mostrará as alterações em tempo real.
