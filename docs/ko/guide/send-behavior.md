# 전송 동작

::: info
**참고**: 이 기능은 버전 1.3.7 이상에서 지원됩니다.
:::

Gemini의 전송 단축키를 `Ctrl+Enter`(macOS에서는 `⌘+Enter`)로 변경하고, `Enter` 키는 줄바꿈으로 사용하여 여러 줄의 프롬프트를 여유롭게 작성할 수 있습니다.

## 이 기능이 필요한 이유

Gemini에서는 기본적으로 `Enter`를 누르면 메시지가 즉시 전송됩니다. 긴 프롬프트를 작성하거나, 여러 줄의 코드를 붙여넣거나, 입력을 세심하게 포맷할 때 오발송이 자주 발생합니다——줄만 바꾸려 했는데 메시지가 이미 전송되어 버리는 것입니다.

활성화하면:

- **Enter** → 줄바꿈 삽입
- **Ctrl+Enter** (macOS: **⌘+Enter**) → 메시지 전송

## 활성화 방법

1. Voyager 설정 패널(팝업)을 엽니다.
2. **Ctrl+Enter 전송** 토글을 찾습니다.
3. 켜면 됩니다.

## 기능 특징

- **제로 성능 오버헤드**: 비활성화 시 키보드 이벤트 리스너가 전혀 동작하지 않아 페이지 성능에 영향이 없습니다.
- **즉시 전환**: 설정에서 토글 후 즉시 반영되며 페이지 새로고침이 필요 없습니다.
- **편집 모드 호환**: 이전에 보낸 메시지를 편집할 때에도 작동합니다.
