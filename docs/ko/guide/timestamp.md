# 메시지 타임스탬프

::: info
**참고**: 이 기능은 버전 1.3.7 이상에서 지원됩니다.
:::

Voyager는 각 메시지의 전송 시간을 자동으로 기록하여 대화의 전체 흐름을 추적할 수 있도록 도와줍니다.

## 기능 소개

Gemini는 기본적으로 메시지 전송 시간을 표시하지 않습니다. 대화가 길어지면 특정 논의가 언제 이루어졌는지 기억하기 어려워집니다.

Voyager는 각 메시지 옆에 정확한 타임스탬프(형식: `YYYY-MM-DD HH:mm:ss`)를 자동으로 표시하여 타임라인을 한눈에 파악할 수 있게 합니다.

## 기능 특징

- **자동 기록**: 메시지 전송 시 자동으로 시간을 캡처하며, 수동 조작이 필요 없습니다.
- **초 단위 정밀도**: 완전한 날짜와 시간으로 긴 대화 리뷰에 적합합니다.
- **로컬 영구 저장**: 타임스탬프는 로컬 스토리지에 저장되며 페이지 새로고침 후에도 유지됩니다.
- **방해 없음**: 시간 정보가 깔끔하게 표시되어 읽기 경험에 영향을 주지 않습니다.

## 사용 시나리오

- **업무 기록**: 특정 결정이 언제 이루어졌는지 되돌아보기.
- **학습 노트**: 개념이 언제 논의되었는지 표시하여 나중에 복습하기.
- **긴 대화 리뷰**: 며칠에 걸친 심층 대화에서 다른 시간대의 토론 내용을 명확히 구분하기.
