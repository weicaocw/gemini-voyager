# 사용자 메시지 LaTeX 렌더링

::: info
**참고**: 이 기능은 버전 1.3.7 이상에서 지원됩니다.
:::

사용자 메시지의 LaTeX 수식을 자동으로 수학 기호로 렌더링합니다.

## 개요

Gemini™ 채팅에서 LaTeX 구문(예: `$E=mc^2$` 또는 `$$\int_0^1 x^2 dx$$`)을 사용하면 Voyager가 자동으로 감지하여 아름다운 수학 공식으로 렌더링합니다.

### 주요 기능

- **인라인 수식**: `$...$`로 감싸기 (예: `$x^2 + y^2 = r^2$`)
- **디스플레이 수식**: `$$...$$`로 감싸면 중앙 정렬된 블록 표시
- **자동 감지**: 기존 메시지와 새로 보낸 메시지를 자동 처리
- **KaTeX 엔진**: KaTeX를 사용한 고품질 수식 렌더링

## 사용 방법

1. Gemini 채팅 입력란에 LaTeX 구문이 포함된 메시지를 입력
2. 전송 후 수식이 자동으로 수학 기호로 렌더링됩니다
3. 인라인 수식 `$...$`은 텍스트 흐름 안에 표시
4. 디스플레이 수식 `$$...$$`은 중앙에 독립된 줄로 표시

## 구문 예시

| 입력                                      | 설명            |
| ----------------------------------------- | --------------- |
| `$E=mc^2$`                                | 인라인 수식     |
| `$$\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$$` | 디스플레이 수식 |
| `$\alpha, \beta, \gamma$`                 | 그리스 문자     |
| `$\sqrt{x^2 + y^2}$`                      | 제곱근          |

## 참고 사항

::: tip 팁

- 사용자가 보낸 메시지에만 적용됩니다. Gemini 응답의 수식은 Gemini 자체에서 렌더링합니다.
- LaTeX 구문에 오류가 있으면 원본 텍스트가 그대로 표시됩니다.
  :::
