# 글꼴 크기

::: warning
**곧 출시 예정**: 이 기능은 다음 버전에서 이용 가능합니다. 기대해 주세요.
:::

Gemini 채팅 영역의 글꼴 크기를 자유롭게 조정하여 가장 편안한 읽기 경험을 만들어 보세요.

## 기능 소개

글꼴 크기에 대한 선호는 사람마다 다릅니다——더 높은 정보 밀도를 위해 작은 글꼴을 선호하는 사람도 있고, 눈의 피로를 줄이기 위해 큰 글꼴을 선호하는 사람도 있습니다. Voyager를 사용하면 채팅 글꼴 크기를 80%에서 150% 사이에서 자유롭게 조절할 수 있습니다.

## 기능 특징

- **유연한 범위**: 80%–150% 글꼴 크기 조절을 지원하여 대부분의 읽기 요구를 충족합니다.
- **전면 적용**: 사용자 메시지와 모델 응답 모두에 적용되며, 본문, 목록, 테이블, 코드 블록을 포함합니다.
- **코드 블록 최적화**: 코드 블록 글꼴 크기가 자동으로 비례 축소(본문 글꼴 × 0.875)되어 코드와 본문의 시각적 계층을 유지합니다.
- **즉시 반영**: 조정 후 즉시 반영되며 페이지 새로고침이 필요 없습니다.

## 사용 방법

1. Voyager 설정 패널(팝업)을 엽니다.
2. **글꼴 크기** 설정을 찾아 스위치를 켭니다.
3. 슬라이더를 드래그하여 글꼴 크기 비율을 조정합니다——페이지가 실시간으로 변화를 미리 보여줍니다.
